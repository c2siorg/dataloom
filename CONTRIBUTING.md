# Contributing to DataLoom

Thank you for considering contributing to DataLoom! Follow these steps to get started:

## How to Contribute
1. Fork the repository.
2. Clone your fork and create a new branch.
3. Make changes and commit them with a clear message.
4. Push your changes and create a pull request.

## Code Guidelines
- Follow best practices for React.js and FastAPI.
- Ensure code is well-documented and follows the project's style guide.

## Reporting Issues
If you find a bug, please open an issue and provide details like steps to reproduce.

## Need Help?
Join our Slack channel: `#dataLoom`
